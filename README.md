- **官方通知群**：`@mkcloudnotice`，单向广播维护通知、活动公告、IP变更等运营信息，订阅人数约6400+

## 二、工单怎么提交：完整步骤与分类建议

工单是MKCloud售后处理的"正规军"。从频道通知里能看出来，无论是免费PUSH机器申请、带宽奖励兑换、临时流量申请、退款申诉、产品过户、实名认证问题，官方都统一要求"提交工单"。这个流程不难，但有些细节会影响处理速度。

**提交步骤：**

1. 登录MKCloud官网客户中心（未注册需先用国内手机号注册并完成实名认证）
2. 进入"客户中心 → 服务单 → 送出服务单"
3. 选择对应的产品或问题分类
4. 标题写清楚问题类型，比如"沪日1TB套餐 端内延迟异常申请协助"而不是笼统的"有问题"
5. 正文详细描述：产品名、出现时间、具体现象、延迟数据或速度截图
6. 提交后等待回复，可在"服务单"列表查看处理进度

**不同问题的工单建议：**

- **技术类问题**（延迟异常、UDP不通、IP变更后连不上）：必须附上具体数据。官方在购物车页面明确写过，如果认为质量存在问题需要退款，要"详细说明具体的问题，如准确具体的延迟数据、速度数据以证明是产品服务质量问题"。技术工单同理，没数据基本等于没说。
- **活动奖励兑换**（带宽奖励、盲盒、IPv6主机）：活动结束后统一提交工单申请，官方会人工审核，禁止小号参与
- **临时流量申请**：节假日期间官方会开放免费临时流量福利，提交工单即可，按顺序处理，不用重复提交
- **产品过户/免费PUSH**：发送方或接收方任一方提交工单申请即可

## 三、实名认证卡住了怎么办：流程与客服协助

实名认证是MKCloud绕不过去的一道坎。因为产品定位是合规跨境电商专线，禁止机场、回国等违法用途，所以从注册开始就要走三重验证。

**实名认证流程：**

1. 手机验证码验证（必须是国内手机号）
2. 身份证信息核验
3. 邮箱验证

官方在隐私政策里写明，处理时效为"简单请求1个工作日内完成，复杂情况延长至7日告知进展"。也就是说，正常情况下24小时内能过，碰到身份信息核验异常或者风控触发，最多7天会给你个进展反馈。

**常见卡点与解决办法：**

- **身份证信息被判定异常**：先检查是不是填错或者用了他人身份信息——官方明确禁止使用非本人身份信息，会被视为欺诈，直接中止服务且不退款
- **手机号收不到验证码**：换运营商或换时段重试，仍然不行就提工单
- **认证一直审核中**：超过1个工作日没动静，提交工单催办，附上注册邮箱
- **省份白名单问题**：专线产品采用省级白名单，只允许设置的一个省份IP连入。如果换了使用地点，需要工单申请修改白名单省份

客服在实名认证环节的作用主要是催办和异常处理，认证本身的核验是系统自动+人工复核，客服没法直接给你"加急通过"。

## 七、退款政策与售后保障：能退吗？怎么退

这是售后里最容易产生纠纷的环节。MKCloud的退款政策写得比较严格，买之前一定要看清楚。

**退款规则核心要点：**

- **24小时窗口**：仅允许产品开通后24小时内、因质量问题申请退款
- **不支持换机房**：服务开通后不支持更换到其他地域的产品
- **质量问题需举证**：需详细说明具体问题，提供准确的延迟数据、速度数据等证明
- **退款后限制**：处理退款后，账户可用于后续订单，但不再有资格获得进一步退款

**不退款的情况：**

- 明显的欺诈行为
- 有意的服务滥用
- 成为DDoS攻击目标
- 用于回国/机场等违规违法业务
- 任何形式违法违规业务

**正常运行时间保证**：官方承诺99.9%正常运行时间（特殊说明的除外）。遇到维护或故障，官方通常会在通知群公告，并根据影响程度给予补偿——比如之前上云互联产品延迟开机，官方统一给全部客户顺延3天使用时长；广港/上云互联产品出现故障时也下发过1天补偿时长。

**退款申请流程：** 在24小时内提交工单，说明质量问题并附数据证据，客服审核后按原支付方式退款。如果觉得退款被拒不合理，可以在工单里补充更多证据继续沟通。

**1. 先查通知群再提工单**

很多"问题"其实是已知的维护或IP变更。官方在`@mkcloudnotice`通知群里会提前公告维护时间、影响范围、IP更换情况。提工单之前先去通知群搜一下关键词，大概率能找到答案，省得你工单排队、客服也省得重复回复。

"我的机器连不上"这种工单，客服只能反问你一堆问题，来回几轮一天就过去了。直接写"沪日1TB套餐，入口IP 172.30.4.x，出口IP变更后SSH连不上，ping延迟正常，telnet 22端口超时"，客服一看就知道往哪个方向排查。

**3. 区分咨询和售后**

买之前问产品选型、线路区别，找TG客服`@mmkkcclloouudd`更快；买之后遇到技术问题、退款、补偿，走工单更稳。两个渠道不要混着用，不然信息散在两边，处理起来更慢。

**4. 节假日关注临时福利**

MKCloud经常在节假日开放工单申请临时流量、免费PUSH、带宽奖励等福利。五一、新春、双旦这些节点，留意通知群公告，符合条件就及时提工单申领，过了活动期就没了。

**5. 实名认证用本人真实信息**

这条不是客服技巧，但能避免你后续找客服处理账号被封的问题。官方对身份信息造假的判定是"欺诈"，直接中止服务不退款。用本人真实信息认证，后续就算要过户、改白名单，工单处理也顺畅。
